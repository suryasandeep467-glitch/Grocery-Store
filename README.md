# Project for Grocery Store Management

## Synopsis
This project is a SQL-based Grocery Store Management system that controls items, suppliers, customers, and orders. It demonstrates the use of various SQL principles like joins, views, stored procedures, and data analysis.

## Characteristics
Add, modify, and remove items
Monitor suppliers and stock levels
Oversee client orders
- Generate sales and stock reports
Utilizing stored procedures, views, and triggers

## Database
MySQL is utilized.
Tables: Orders, Customers, Suppliers, Products, etc.
Relationships: Foreign and Primary Keys
Examples of analytics and reporting queries
